Description
-----------
A gtk+ application that performs the following transformations to a polygon defined by a set of coordinates:
 + scaling
 + rotate
 + shearing
 + translation

Dependencies
------------
 + gtk 2.0 (libgtk2.0-dev on Ubuntu)

How to compile and run
----------------------
	make
	./form-shifter
